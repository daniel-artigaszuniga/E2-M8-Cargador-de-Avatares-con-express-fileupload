# E2-M8 - Cargador de Avatares con express-fileupload

## Descripción
Este proyecto corresponde al ejercicio **Cargador de Avatares con express-fileupload** del módulo 8.

La aplicación fue desarrollada con **Node.js** y **Express**, utilizando el middleware **express-fileupload** para gestionar la subida de archivos.

El objetivo del ejercicio es implementar un endpoint que reciba una imagen, valide su extensión y la guarde en el servidor con un nombre estandarizado basado en el ID del usuario.

---

## Tecnologías utilizadas
- Node.js
- Express
- express-fileupload
